# Employee Attendance System - Face Recognition

## 📋 System Requirements
- Python 3.8 or higher
- MySQL Server 8.0 or higher
- Webcam
- 4GB RAM minimum

## 🚀 Quick Start

### Windows:
1. Double-click `build.bat`
2. Follow the on-screen instructions

### Linux/Mac:
1. Run: `chmod +x build.sh`
2. Run: `./build.sh`
